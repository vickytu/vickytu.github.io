== RADIATE ==

Radiate is a clean, creative and light retina ready responsive blogging theme supporting parallax header image. It integrates primary color option, custom background, custom css and few other features through WordPress customizer. The theme is translation ready and you can get free support on http://themegrill.com/forums.

= Theme Features Usage =
All available options can be used from Appearance->Customize

Q. How to set the featured section of Front page?
A. In the dashboard, go to Appearance->Customize. Click on 
Front Page Featured Section and select the pages from the dropdwon. 
Images used for these pages in front section are the featured images of those respective pages.

== IMPORTANT NOTES ==
* This theme supports post thumnail but are not used for posts but are used for pages that are
  used in the featured section of front page. 
* This theme does not support a place for the theme tagline to be shown in front end.
